# Getting started

To use most API endpoints you need to provide a bearer token in the `Authorization` header of you requests. For this you can use an API key as the token.

## Get your API Key

1. Sign in at [vision-x-dataspace.base-x-ecosystem.org](https://vision-x-dataspace.base-x-ecosystem.org)
2. Generate an API key using the button in the upper right corner
3. Copy and save the generated API key somewhere

## Validity

The API key has no expiration date. However, the full key will only be visible once upon generating. Additionally, generating a new key revokes any previous ones.
